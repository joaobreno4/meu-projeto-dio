# 🚀 Projeto Prático DIO

Este repositório foi criado como parte de um desafio prático da [DIO](https://www.dio.me/), com o objetivo de aplicar na prática os conhecimentos adquiridos e fortalecer o portfólio no GitHub.

---

## 📌 Sobre o Projeto
O projeto tem como foco a organização de um ambiente completo de desenvolvimento, incluindo código, documentação, estrutura de diretórios e versionamento utilizando Git e GitHub.

---

## 🎯 Objetivos
- Aplicar conceitos aprendidos durante a formação
- Criar um projeto estruturado e profissional
- Utilizar Git e GitHub de forma prática
- Melhorar o portfólio para o mercado de trabalho
